Last Update: 2014/10/9

Tweet
SimcirJS is fully rewrited version of Simcir (circuit simulator) with HTML5, JavaScript and jQuery.

Usage

  • Choose a device from the toolbox and move to right side.
  • Connect them by drag operation.
  • Click an input node to disconnect.
  • Move a device back to the toolbox if you don't use.
  • Ctrl+Click to toggle view (Live circuit or JSON data)
