DevPost

Download

DevPost is a tiny smtp server for developers.
The real smtp server rejects mail like a spam for security reasons.
Although it's a matter of course, is not useful when try to send a mail while testing.
DevPost accepts all the mails without exception.

Setup

  1. Download a module and deploy to an application server.
  2. Make a mailbox directory named '/tmp/devpost-mbox'.
  3. Send test mail to HOSTNAME:2525. (mailbox directory and port number can change by web.xml)
  4. Open a web browser and access to http://HOSTNAME:HTTPPORT/devpost.
